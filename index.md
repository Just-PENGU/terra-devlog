# Terra – Devlog

**Terra** is a voxel sandbox game, inspired by Minecraft, created by **PENGU**.

## Alpha 1.0 – Implemented Features
- Grass Block
- Infinite terrain
- Place and break blocks
- Highlighted selected block
- Fully functional hotbar
